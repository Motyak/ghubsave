# HOW TO INSTALL
```console
$ sh <(curl -sS https://raw.githubusercontent.com/Motyak/ghubsave/master/install.sh)
```

# HOW TO USE
e.g.: Clone the 10 most recently updated repo from user Motyak and archive it in current directory :
```console
user@host:~/backup$ ghubsave Motyak . 10
```
